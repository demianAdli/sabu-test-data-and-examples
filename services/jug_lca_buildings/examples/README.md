# `jug_lca_buildings` Examples

This folder contains examples for using the `jug_lca_buildings` service in
three ways. Instructions are available in the README file for each
corresponding directory:

- `direct_python/`: run the service directly from Python.
- `api/`: run the Flask API locally and call it with `curl` or Postman.
- `docker/`: run the service from the published Docker image.

For Python-based examples, install the service package and its runtime API
dependencies in the Python environment you use for the examples. At minimum,
this includes `jug_lca_buildings`, `sabu_chassis`, `flask`, and
`flask-smorest`.
