# `jug_lca_buildings`

This folder contains test data, examples, results, and benchmark materials for
the `jug_lca_buildings` service in the Sabu framework.

`jug_lca_buildings` estimates embodied and end-of-life carbon emissions for
building GeoJSON inputs. The materials here are meant for trying the service,
checking expected inputs and outputs, and reproducing selected runs.

## Where to Look

- `datasets/`: input GeoJSON datasets and links to larger Zenodo datasets.
- `examples/`: instructions for running the service with direct Python, a local
  API, or Docker.
- `results/`: example outputs and links to reproducibility result datasets.
- `benchmarks/`: performance logs, timing summaries, and benchmark notes.
- `docs/`: additional service-specific documentation, when available.

For a first run, start with `examples/README.md`, then choose the execution mode
that matches your environment.
