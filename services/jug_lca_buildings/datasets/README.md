# `jug_lca_buildings` Datasets

This folder contains GeoJSON datasets for buildings in Montreal to be used as input for `jug_lca_buildings`.

Two files with 1 and 50 buildings are also available in the current folder as samples.

## Montreal Buildings GeoJSON Dataset

This dataset contains GeoJSON files for buildings in Montreal, with sample sizes ranging from 1 to 500 buildings.

- Record: https://zenodo.org/records/19501061
- DOI: https://doi.org/10.5281/zenodo.19501061
- Version: `v2`

Citation:

Adli, A. (2026). _Montreal Buildings GeoJSON Dataset_ [Data set]. Zenodo. https://doi.org/10.5281/zenodo.19501061

## Large-Scale Building GeoJSON Dataset

This dataset contains GeoJSON files for buildings in Montreal, with sample sizes ranging from 1000 to 30000 buildings.

- Record: https://zenodo.org/records/19500482
- DOI: https://doi.org/10.5281/zenodo.19500482
- Version: `v2`

Citation:

Adli, A. (2026). _Large-Scale Building GeoJSON Dataset_ [Data set]. Zenodo. https://doi.org/10.5281/zenodo.19500482
