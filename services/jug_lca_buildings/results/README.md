# `jug_lca_buildings` Results

This folder contains example output files and processed result tables generated by
the `jug_lca_buildings` service.

## Embodied and End-of-Life Carbon Emissions Dataset for 30,000 Montreal Buildings

This record includes GeoJSON files for buildings in Montreal, containing 1000 to
30000 buildings to be used as input for `jug_lca_buildings`.

- Record: https://zenodo.org/records/19885627
- DOI: https://doi.org/10.5281/zenodo.19885627
- Version: `v1.0.0`

Citation:

Adli, A. (2026). _Embodied and end-of-life carbon emissions dataset for 30,000
Montreal buildings_ (v1.0.0) [Data set]. Zenodo.
https://doi.org/10.5281/zenodo.19885627

## Contents

The Zenodo record should contain the reproducibility outputs for
`jug_lca_buildings`, including generated life-cycle carbon estimation results and
any processed tables used for validation, comparison, or benchmarking.

## How to Use

Download the result files from the Zenodo record and place them in this folder
when reproducing the examples or validating service outputs locally.

The expected folder structure depends on the files included in the Zenodo
archive. Preserve the archive's internal directory structure unless a specific
example or script states otherwise.
